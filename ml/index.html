Detect key points of the human body.

  • README
  • single: works with normalised key points, smoothing them and visualising on a canvas. Assumes a single source and single pose.
  • multiple: receive & process multiple poses from any number of sources
  • geometry: calculates geometric relations between parts of the body
  • gradient: Use horizontal centroid of poses to manipulate gradient stops.
